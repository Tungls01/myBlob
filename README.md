myBlob

A full-featured dApp to upload, download, manage, and view blobs on Shelby, the high-performance decentralized storage protocol on Aptos.

Features:
Upload multiple blobs at once
View all blobs directly in the dapp: video, image, music, text, etc.
Search blobs by Aptos wallet address
Full interaction with ShelbyNet

Upcoming Features:
AI integration to enhance blob management and retrieval


Live: https://my-blob-tau.vercel.app
